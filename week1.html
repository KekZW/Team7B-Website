Main Insights

Key insights from the partner meeting.

  • Real-time Data: The need for real-time water monitoring to ensure safety during river activities.
  • Pollution Awareness: Importance of providing pollution data to the public to raise awareness.
  • Minimum Cost: The solution should be cost-effective, considering the non-profit nature of the organization.
  • Easy to maintain and understand: The solution should be user-friendly and easy to maintain, ensuring long-term sustainability.

Brainstorming Ideas

River Information Display System

Install digital Information Display System, similar to those at the beaches or bus stop that display real-time information about the rivers.

QR Code + Website

Implement a QR code system that links to a website providing real-time river data and alerts.

Mobile Application

A mobile app that provides real-time updates on river conditions, safety alerts, and personalized activity suggestions.

Final Chosen Solution

After evaluating all ideas from our brainstorming session, we selected the most practical and impactful solution to move forward with.

Mobile Application

The mobile application was chosen as it provides a comprehensive solution for real-time water monitoring and activity planning. It allows users to access live data, receive alerts, and get personalized activity suggestions based on current river conditions and possible weather conditions.

We chose this solution as compared to the other solutions, this is the most practical, feasible and most importantly cost effective solution.
